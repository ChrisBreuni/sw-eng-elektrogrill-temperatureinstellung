# 1. Funktionale Requirements
## F1.1 Wunschtemperaturanzeige
- Die gewünschte Temperatur des Elektrogrills wird angezeigt und befindet sich in einem
Messbereich von 50 bis 500 °C.
- Es sollte dem Benutzer klar sein, auf welcher Anzeige die Wunschtemperatur zu sehen ist.
## F1.2 Wunschtemperaturänderung
- Die Wunschtemperatur kann im Messbereich von 50 bis 500 °C geändert werden.
## F2. Aktuelle Temperatur
- Die aktuelle Temperatur des Elektrogrills wird angezeigt und befindet sich in einem
Messbereich von 50 bis 500 °C.
- Es sollte dem Benutzer klar sein, durch was die aktuelle Temperatur angezeigt wird.
## F3. Zieltemperaturstatus
- Der Elektrogrill hat eine Anzeige, die dem Benutzer andeutet, ob die 
Zieltemperatur bereits erreicht wurde.
- Die Anzeige des Zieltemperaturstatus sollte leicht gehalten sein.
## F4. An-Aus-Funktion
- Der Elektrogrill hat eine Funktion, den Grill an- und auszuschalten.
- Der Benutzer sieht eindeutig, ob der Grill an- oder ausgeschaltet ist.
## F5. Resttemperaturanzeige
- Der Elektrogrill hat eine Anzeige, die dem Benutzer zeigt, dass der Grill aus
aber noch nicht vollständig abgekühlt ist.

# 2. Nicht-funktionale Requirements
## NF6.1 Programmiersprache
- Das Projekt des Elektrogrills wird in der Programmiersprache Python entwickelt.
## NF6.2 Tools & Packages
- Das Projekt benutzt das Python package tkinter, um das Interface bereitzustellen.
